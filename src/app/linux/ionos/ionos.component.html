  • The following instructions will allow for self hosting with IONOS
  • First install the utility:
  • pip install domain-connect-dyndns
  • domain-connect-dyndns setup --domain
  • Then update using:
  • domain-connect-dyndns update --all
  • To add a cron job try this:
  • crontab -e
  • */1 * * * * /usr/bin/flock -n /tmp/ipupdate.lck /usr/local/bin/domain-connect-dyndns update --all --config /root/dyndns/settings.txt
